Zoo Animals

As a student project, I created a simple website that would be used to help a new employee at the zoo become more familiar with the animals there. In this project, I learned how to put images on a website and manipulate the scale of the image. We also learned how to link to another website with an anchor tag, which I set to open the link in a new tab. I experimented with roll over text for the images which would indicate the name of the animal in the picture.
